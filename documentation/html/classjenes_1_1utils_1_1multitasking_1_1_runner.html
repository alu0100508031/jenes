Detailed Description

The Runner is the abstraction of an execution enviroinment per Jenes. By now this approach has been used to parallelize the fitness evaluation step. The programmer have to override the default call-backs than use execute(jenes.GeneticAlgorithm) or execute(jenes.GeneticAlgorithm, boolean) to activate Genetic Algorithm to start evolving in the enviroinment.

2.0
