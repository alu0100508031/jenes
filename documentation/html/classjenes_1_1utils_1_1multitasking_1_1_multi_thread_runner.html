Constructor & Destructor Documentation

jenes.utils.multitasking.MultiThreadRunner.MultiThreadRunner ( )

Default constructor. This will instantiate a MultiThreadRunner envoiroinment wich thread number is equals to the maximum number of processor phisical avaible to the host Java Virtual Machine.
This approach minimize the overhead in managing the thread because the latest JVMs could try to dispach each thread on a different phisical processor obtaining the best speed-up in performances.
